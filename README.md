# benri-na-pdf

スキャンPDFの編集に向いた、ブラウザ技術 (Electron) ベースの軽量 PDF エディタ。

- 公式サイト: <https://informanellica.github.io/benri-na-pdf/>
- マニュアル: <https://informanellica.github.io/benri-na-pdf/manual.html>
- 更新履歴: <https://github.com/informanellica/benri-na-pdf/releases>
- 最新版ダウンロード: <https://github.com/informanellica/benri-na-pdf/releases/latest>

## 主な機能

- PDF 表示 (単ページ / 連続スクロール / 見開き / フィット)
- 二値化 (adaptiveMean / Gaussian / Otsu / 単純) ・傾き補正・CLAHE 前処理
- OCR (日本語・英語) と検索可能 PDF への埋め込み
- ページ操作 (回転・削除・並べ替え・コピー&ペースト・サムネイル右クリック)
- 編集ツール (テキスト・描画・ハイライト・コメント)
- 結合 (PDF + 画像) ・圧縮保存 (DPI / JPEG 品質指定)
- AI 要約 (Ollama / OpenAI / Anthropic / Gemini / Mistral)
