プライバシーポリシー

最終更新日: 2026-05-29

Informanellica (以下「当方」といいます) は、本ソフトウェア「benri-na-pdf」(以下「本ソフトウェア」といいます) の 利用に関し、本プライバシーポリシーに従って利用者の情報を取り扱います。

1. 当方が収集する情報

本ソフトウェアは、当方に対して利用者の個人情報、利用状況、操作履歴等を 一切送信しません。 テレメトリ、利用統計、クラッシュレポート、エラー報告等の自動送信機能は実装されていません。

2. 利用者のコンピュータに保存される情報

本ソフトウェアは、以下の情報を利用者のコンピュータ内 (ブラウザのローカルストレージおよび設定ファイル) に保存します:

  • アプリ設定 (二値化パラメータ、表示モード、ダークモード等)
  • 最近開いたファイルの履歴 (ファイルパスのみ。ファイル内容は含みません)
  • ツールバーの並び順
  • AI 要約機能を利用する場合に利用者自身が入力した API キー

これらは 利用者のコンピュータから外部へ送信されることはありません。

3. 第三者サービスへの送信

本ソフトウェアの「AI 要約」機能を利用者が明示的に有効化し実行した場合に限り、 要約対象の PDF テキストおよび利用者が設定した API キーが、利用者が選択した AI プロバイダ (OpenAI / Anthropic / Google Gemini / Mistral / Ollama 等) に 直接送信されます。

  • 送信は利用者の操作によって発生し、当方を経由しません。当方はこれらの通信内容を閲覧・保存・転送できません。
  • 送信先 (AI プロバイダ) における取り扱いは、各プロバイダのプライバシーポリシーに従います。
  • Ollama 等のローカル LLM を選択した場合、送信先は利用者自身が用意したローカルエンドポイントとなり、インターネットへの送信は発生しません。

4. PDF ファイルの取り扱い

利用者が本ソフトウェアで開いた PDF ファイル、編集内容、OCR 結果、二値化結果等は すべて利用者のコンピュータ内で処理されます。 これらが当方や第三者に自動送信されることはありません (前項の AI 要約機能で利用者が明示的に送信操作を行った場合を除きます)。

5. クッキー・トラッキング

本ソフトウェアはデスクトップアプリケーションであり、トラッキングクッキーや広告識別子の使用はありません。

6. 子どもの情報

本ソフトウェアは、子どもからの個人情報を収集することを意図していません。前述のとおり、当方は利用者からいかなる個人情報も収集しません。

7. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、 GitHub Issues までお願いします。

8. 改訂

本プライバシーポリシーは予告なく変更される場合があります。 変更後の内容は本ページに掲載された時点で効力を発生します。
