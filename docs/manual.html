インストールと起動

  1. 最新版リリースから benri-na-pdf Setup x.y.z.exe をダウンロード
  2. インストーラを実行 (インストール先・スタートメニュー登録を選択可能)
  3. スタートメニューから「benri-na-pdf」を起動

初回起動時はチュートリアルが表示されます。後からも右上の ボタンで再表示できます。

ファイル操作

  • PDFを開く — ツールバー左端の ボタン、またはウィンドウへ PDF をドラッグ&ドロップ
  • ファイル結合 — ボタン。複数 PDF・画像を選択 → プレビューモーダルで順序確認 → 「結合して開く」または「結合して保存」
  • 最近開いたファイル — ホーム画面に最近のファイルが並びます。クリックで再オープン
  • 関連付け — インストール後、PDF を右クリック →「プログラムから開く」で benri-na-pdf を選択できます

表示モード

ツールバー中央の表示モードボタンで切替:

  • 単ページ — 1 ページずつ表示
  • 連続スクロール — 全ページを縦に並べてスクロール
  • フィット — ウィンドウサイズに合わせて表示
  • 見開き — 2 ページ並べて表示。右開き(縦書き)/左開き(横書き)を切替可能

ページ操作

  • ページ移動 — ページ番号入力欄、または ←/→ キー、マウスホイール (単ページ時)
  • サイドバーサムネイル — クリックで該当ページにジャンプ、ドラッグ&ドロップで並べ替え
  • サムネイル右クリックメニュー — 削除・コピー・切り取り・貼り付け・回転・二値化・傾き補正などを直接実行
  • 複数選択 — Ctrl+クリックでトグル、Shift+クリックで範囲選択、Ctrl+A で全選択
  • 回転 — ツールバーの / で 90 度ずつ回転
  • 削除 — Delete キーまたは ボタン
  • コピー&貼り付け — Ctrl+C / Ctrl+V

編集ツール

  • テキスト — クリックでテキストボックスを追加、ドラッグで移動、ダブルクリックで編集
  • 描画 — フリーハンドで線を描画。色・太さを調整可
  • ハイライト — 範囲をドラッグして黄色いマーキング
  • 消しゴム — 描画やハイライトを消去
  • コメント — 付箋のような注釈をページに貼付

編集内容は保存時に PDF へ焼き込まれます。

画像処理 (二値化・傾き補正)

二値化

  • 選択ページを二値化 — サイドバーで複数選択していればそれら、無ければ現在ページのみ
  • すべて二値化 — 全ページを一括で二値化

パラメータはツールバー右端の アプリ設定から変更できます。

  • アルゴリズム — adaptiveMean (デフォルト・高速) / adaptiveGaussian (高品質・低ノイズ) / Otsu (自動) / 単純しきい値
  • blockSize — 局所平均の窓サイズ。21〜51 が目安。大きいほど滑らかに
  • C — オフセット。小さくすると薄い画素が残りやすい (例: 3〜10)
  • CLAHE 前処理 — 局所コントラスト強調
  • モルフォロジー閉じ処理 — 細線の途切れを補修 (太る副作用あり)

傾き補正

ボタンで Hough 変換ベースの自動傾き検出 → 補正。スキャン文書を真っ直ぐに揃えます。

OCR

  • OCR (JP) — 日本語文字認識
  • OCR (EN) — 英語文字認識

結果モーダルで:

  • コピー — テキストをクリップボードへ
  • PDFに埋め込む — 透明テキストレイヤとして PDF に埋め込み (保存時に適用)。検索可能 PDF になります

保存と圧縮保存

  • 名前を付けて保存 — OS の保存ダイアログで保存先を指定
  • 上書き保存 (Ctrl+S) — 開いていたファイルに上書き (初回は保存ダイアログ)
  • 圧縮保存 — 解像度 (DPI) と JPEG 品質を指定して再エンコード保存。メール添付サイズに調整したいときに

AI 要約

ツールバーの で右サイドバーを開き、現在ページまたは全ページの要約を AI に依頼できます。プロバイダーは設定アイコンから選択:

  • Ollama (ローカル) — オフライン動作。事前に ollama pull モデル名 でモデル取得が必要
  • OpenAI / Anthropic / Google Gemini / Mistral — オンライン。API キー要

API キーはブラウザのローカルストレージに保存され、外部送信は AI プロバイダーへの API リクエストのみです。

キーボードショートカット

Ctrl+S上書き保存
Ctrl+A全ページ選択
Ctrl+C選択ページをコピー
Ctrl+X選択ページを切り取り
Ctrl+Vページを貼り付け
Delete選択ページを削除
F12 / Ctrl+Shift+I開発者ツール開閉

アプリ設定

ツールバー右の ボタンから設定モーダルを開きます。設定はブラウザのローカルストレージに保存され、次回起動時にも維持されます。

  • 二値化パラメータ — アルゴリズム・blockSize・C・CLAHE・モルフォロジー処理
  • 既定値に戻す — モーダル左下のボタン
